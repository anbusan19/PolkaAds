# Fonts Directory

Place the following font files in this directory:

## Geist
- `GeistVF.woff2` - Variable font (weights 100-900)
- Download from: https://vercel.com/font

## PP Neue Montreal
- `PPNeueMontreal-Regular.woff2` (weight 400)
- `PPNeueMontreal-Medium.woff2` (weight 500)
- `PPNeueMontreal-Bold.woff2` (weight 700)
- Purchase from: https://pangrampangram.com/products/neue-montreal

## PP Supply Sans
- `PPSupplySans-Regular.woff2` (weight 400)
- Purchase from: https://pangrampangram.com/products/supply

## Inter (Fallback)
Inter is loaded from Google Fonts as a fallback and doesn't require local files.

---

**Note:** Make sure you have the proper licenses for these fonts before using them in production.

## Alternative: Using System Fonts

If you don't have access to these fonts, the application will fall back to system fonts:
- Geist → system-ui
- PP Neue Montreal → system-ui
- Inter → Google Fonts (already configured)
- PP Supply Sans → Courier New, monospace
